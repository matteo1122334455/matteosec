# matteosec
MatteoSec Live 0.1 Alpha é a primeira versão pública do projeto. Ainda é uma versão inicial, mas já conta com boot híbrido BIOS/UEFI, XFCE automático, wallpaper próprio e ferramentas para estudo, laboratório, CTF e defesa.
## Usuários e senhas

O MatteoSec Live inicia automaticamente no usuário padrão:

Usuário: `matteo`  
Senha: `live`

A conta root também está habilitada:

Usuário: `root`  
Senha: `toor`
